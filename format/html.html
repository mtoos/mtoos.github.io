About HTML Formatter / Minifier

Free online tool to format an ugly HTML code, making it readable and pretty.

How do I format HTML code?

  • Paste your HTML into the input box.
  • Click "Format" or "Compress" button,you will get formatted or compressed HTML in the same input box.
  • Click "Copy to clipboard" button copy the formatted HTML to your clipboard.
  • Your HTML data is secure. The HTML formatter does not send your HTML data to our servers for formatting, the process is done completely in your web browser.

What is HTML?

HTML, in full hypertext markup language, a formatting system for displaying material retrieved over the Internet.HTML markup tags specify document elements such as headings, paragraphs, and tables. They mark up a document for display by a computer program known as a Web browser.
